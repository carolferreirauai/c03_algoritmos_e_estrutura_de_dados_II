## 💻 Algoritmos e Estrutura de Dados II (C03) - Inatel

### 📌 Sobre o Repositório

Este repositório contém todos os códigos, listas de exercícios e trabalhos práticos desenvolvidos durante a disciplina **C03 - Algoritmos e Estrutura de Dados II**, cursada no **Instituto Nacional de Telecomunicações (Inatel)**.

O objetivo da disciplina é o estudo da análise de complexidade de algoritmos, algoritmos de busca e ordenação, recursão, hashing, alocação dinâmica de memória e estruturas de dados avançadas.

* **Aluna:** Caroline Ferreira de Souza

### 🚀 Tecnologias Utilizadas

* **Linguagem:** C++
* **Ambiente de Desenvolvimento:** VS Code

### 💡 Como Executar

Para compilar e rodar os códigos deste repositório:

1. Clone o repositório:
   `git clone https://github.com/carolferreirauai/c03_algoritmos_e_estrutura_de_dados_II.git`

2. Navegue até a pasta do arquivo desejado via terminal.

3. Compile o código:
   `g++ nome-do-arquivo.cpp -o executavel`

4. Execute o programa:
   `./executavel` (Linux/Mac) ou `executavel.exe` (Windows).
